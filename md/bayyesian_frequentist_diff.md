# 频率学派与贝叶斯学派优劣对比？

频率学派以频率近似概率为核心思想，在简单事件的概率分析中，合适高效，如掷骰子，赌博等问题，但对于复杂的实际问题就不适用，如专家系统中的推理，
涉及多重复杂因素等。而贝叶斯学派可以解决频率学派无能为力的推理问题，并且贝叶斯学派将概率本身视为变量，将该变量的超参独立出来，在后来机器学习算法的
实践检验中，证明贝叶斯学派的此种概率变量化观点是科学而适用的。

# 频率学派与贝叶斯学派联系
频率学派可以看做超参均匀的贝叶斯学派。贝叶斯学派的计算方法在样本足够多的情况下近似于频率学派，可以说样本越多，二者越近似等。
